# techero_problem2
This program collects payments from customers using Raves payment gateway and the payment is share between the compay and the drivers. 

## How to use it
Firstly, install python 3.6 which comes with the package manager pip. Then proceed to downloading all the files in requirements.txt.

When done clone the repo using 
```
git clone <repo>
```
this clones the repo to your local machine.

Go to you command prompt or terminal (linux users) and type 
```
py manage.py runserver
```
to run the server and you can then view the page at http//:127:8000.
